# Next.js with Paddle.com checkouts example

This example shows how to use [Paddle.com checkouts](https://paddle.com/) in Next.js along with [react-bootstrap](https://react-bootstrap.github.io/).

## Preview

Preview the example live on [StackBlitz](http://stackblitz.com/):

[![Open in StackBlitz](https://developer.stackblitz.com/img/open_in_stackblitz.svg)](https://stackblitz.com/github/dropkickfish/paddle-nextjs)

## Deploy your own

Deploy the example using [Vercel](https://vercel.com?utm_source=github&utm_medium=readme&utm_campaign=next-example):

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/git/external?repository-url=https://github.com/dropkickfish/paddle-nextjs)

## How to use

Execute ```npm run dev``` or ```yarn dev``` to run in dev mode.

Deploy it to the cloud with [Vercel](https://vercel.com/new?utm_source=github&utm_medium=readme&utm_campaign=next-example) ([Documentation](https://nextjs.org/docs/deployment)).
